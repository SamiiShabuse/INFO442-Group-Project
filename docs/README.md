# Docs Folder

This folder contains project planning notes and supporting documentation.

## Current Documents

- `project_scope.md` explains the project idea, goals, and framing.
- `why_this_project_works.md` explains why the project is feasible and useful.
- `weekly_summary.md` tracks weekly progress.
- `class_syllabus.md` stores course context and requirements.

## What Belongs Here

- Planning notes
- Scope documents
- Weekly summaries
- Project decisions and rationale
- Supporting class or process documents

## What Does Not Belong Here

- Raw or processed datasets
- Notebook experiments
- Reusable Python code
- Dashboard app files
- Final polished reports or slides

Use `reports/` for final deliverables and `notebooks/` for executable analysis.
