# Raw Data

This folder contains original source-level data files before project cleaning and integration.

## Folders

- `fred/` contains raw FRED macroeconomic and risk series.
- `wikipedia/` contains raw S&P 500 constituent metadata.
- `yfinance/` contains raw Yahoo Finance market data.

Raw files should stay close to the original downloaded format. Cleaned files belong in `data/processed/source_data/`.
