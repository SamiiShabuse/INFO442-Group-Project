# Processed Source Data

This folder contains cleaned datasets that still belong to one source.

## Folders

- `fred/` contains cleaned FRED macroeconomic and risk data.
- `wikipedia/` contains cleaned ticker, company, and sector metadata.
- `yfinance/` contains cleaned price, return, and summary statistics data.

These files are generated by `notebooks/01_source_data/` and are used by `notebooks/02_integration/01_data_integration.ipynb`.
