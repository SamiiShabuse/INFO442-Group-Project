# Modeling Outputs

This folder stores model predictions and evaluation metrics.

## Folder Structure

- `linear_regression/`
- `random_forest/`
- `gradient_boosting/`
- `ridge_regression/`

Each model folder should contain:

- `test_predictions.csv`
- `metrics.csv`

These files are generated by the notebooks in `notebooks/04_modeling/`. Keeping one folder per model makes comparison easier and prevents model outputs from being mixed together.
