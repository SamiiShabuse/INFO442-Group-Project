# Reports Folder

This folder is for polished project deliverables and presentation materials.

## What Belongs Here

- Final report
- Presentation slides
- Polished figures or charts used in the report
- Modeling and evaluation writeups
- Dashboard screenshots for final submission
- Project proposal drafts, if they are part of the final deliverables

## What Does Not Belong Here

- Raw datasets
- Processed CSV outputs
- Notebook scratch work
- Reusable Python source code
- Dashboard app files

Reports should explain the project clearly: the problem, data sources, processing pipeline, modeling methods, evaluation results, limitations, and final conclusions.
