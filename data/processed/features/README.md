# Feature Data

This folder contains the model-ready feature dataset and the results of feature selection.

Generated by:

- `notebooks/03_features/01_feature_engineering.ipynb`
- `notebooks/03_features/02_feature_selection.ipynb`

Expected files:

- `feature_engineered_dataset.csv` — full candidate feature set, used as the input to feature selection.
- `selected_features.csv` — final list of features chosen by feature selection (one feature per row).
- `feature_selection_summary.csv` — ranking table for every candidate feature (target correlation, Random Forest importance, mutual information, and which features were dropped and why).

`feature_engineered_dataset.csv` is used by all notebooks in `notebooks/04_modeling/`. `selected_features.csv` is intended to be the feature list those notebooks load going forward instead of hardcoding it.
